Este é um projeto pessoal que tem como objetivo apresentar camisas de times de futebol que vendo atualmente.

Hoje, as vendas são realizadas principalmente por meio de conversas diretas com clientes. No entanto, estou desenvolvendo este projeto com a ideia de criar um site próprio. Inicialmente, o site pode funcionar apenas como um mostruário das camisas disponíveis, mas futuramente poderá evoluir para um site completo de vendas.

Além da divulgação dos produtos, este projeto também tem como objetivo contribuir para o meu desenvolvimento como programador. Sou estudante de Ciência da Computação e estou utilizando este projeto para praticar conceitos de desenvolvimento web.

Tecnologias utilizadas ou planejadas:

HTML

CSS

JavaScript

Backend em Java (planejado)

Banco de dados (planejado)

Status do projeto: Em desenvolvimento.
